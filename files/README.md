# About CV PDF

Place your CV PDF here: `Jareen_Anjom_CV.pdf`

The file will be automatically available for download at:
`https://jareenanjom.github.io/files/Jareen_Anjom_CV.pdf`

To add it:
1. Save your CV as `Jareen_Anjom_CV.pdf`
2. Place it in this directory (`files/`)
3. Commit and push to GitHub

The download link in the navigation will work automatically once the file is in place.
